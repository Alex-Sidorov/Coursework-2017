
# Личный бухгалтер  
Приложение для ведения личной бухгалтерии 

**Функционал:**
* Запись, хранение и обработка информации о расходах и доходах по дням

* Запись, хранение и обработка информации о долгах и должниках

* История доходов и расходов

 * Хранение информации об общей сумме денег

* Информация о среднем числе доходов и расходов за N  дней

* Конвертер валют

**Интерфейс:**

Интерфейс представлен командной строкой. Перемещение по меню осуществляется через кнопки на клавиатуру: вверх, вниз, Enter, BackSpace.

**Данные:**

Данные хранятся в текстовом файле
